# Parliamentary Motion Classification

## Overview

A machine learning project developed as part of MSc Computer Science coursework. The objective was to classify parliamentary motions into relevant categories using machine learning techniques and text analysis.

## Features

* Data preprocessing and cleaning
* Feature engineering
* Machine learning model training
* Classification and prediction
* Performance evaluation

## Technologies Used

* Python
* Scikit-learn
* Pandas
* NumPy
* Matplotlib

## Project Tasks

* Explored and prepared real-world parliamentary data
* Built and evaluated classification models
* Compared model performance using evaluation metrics
* Analysed ethical considerations of machine learning systems

## Learning Outcomes

* Machine learning workflows
* Data preparation and preprocessing
* Model evaluation and optimisation
* Applied AI and NLP concepts
